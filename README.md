# LLM 知识库

> 用于持续整理 LLM 学习过程中的问题、结论、公式与代码理解。

## 结构说明

- [normalization.md](./normalization.md): 归一化相关知识点
- [positional-encoding.md](./positional-encoding.md): 位置编码相关知识点
- [pytorch-basics.md](./pytorch-basics.md): PyTorch 基础机制
- [template.md](./template.md): 新增条目时使用的统一模板

## 维护原则

- 新条目优先归入已有主题文件。
- 每个条目尽量保持统一结构：`问题`、`核心结论`、`公式`、`解释`、`补充`。
- 如果某个主题条目过多，再单独拆出新文件，比如 `attention.md`、`kv-cache.md`。
